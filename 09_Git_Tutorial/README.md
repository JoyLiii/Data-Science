## 1. 為什麼要學 Git?
學習到現在大家一定累積很多的程式碼或是各式的檔案，如何去做有效的控管其實是非常重要！
想像有個情境是你上線了一段程式碼，但顧客抱怨使用上有問題，你需要緊急還原到上一版，這是就可以
靠 Git 來協助你進行程式碼或檔案版控，快速還原上一版，就不用急忙打電話給同事找程式碼。
我們可以利用 Git 指令將程式碼每次的版本記錄起來，當哪一天需要回頭檢視的時候就可以直接查看。
